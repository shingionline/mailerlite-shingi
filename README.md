## Installation instructions

- git clone https://github.com/shingionline/mailerlite-shingi.git
- cd mailerlite-shingi
- composer install
- npm install
- npm run dev
- php artisan serve
- visit <a href="http://127.0.0.1:8000" target="_blank">http://127.0.0.1:8000</a>
